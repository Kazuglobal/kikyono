# ヒーロー画像の設置方法

ファーストビュー（ヒーローセクション）で使用する画像をこのフォルダに配置してください。

## 必要な画像ファイル

以下の4つの画像ファイルを用意してください：

1. **hero-team.jpg** (フォールバック画像)
   - 推奨サイズ: 1920x1080px
   - 用途: すべてのデバイスのフォールバック画像
   - 形式: JPEG（推奨）

2. **hero-team-mobile.jpg** (モバイル用)
   - 推奨サイズ: 640x480px または 640x1136px
   - 用途: 画面幅640px以下のモバイルデバイス
   - 形式: JPEG（推奨）

3. **hero-team-tablet.jpg** (タブレット用)
   - 推奨サイズ: 1024x768px
   - 用途: 画面幅641px〜1024pxのタブレットデバイス
   - 形式: JPEG（推奨）

4. **hero-team-desktop.jpg** (デスクトップ用)
   - 推奨サイズ: 1920x1080px
   - 用途: 画面幅1025px以上のデスクトップデバイス
   - 形式: JPEG（推奨）

## 画像の最適化のヒント

- **ファイルサイズ**: 各画像はできるだけ小さく圧縮してください（モバイル: 100KB以下、デスクトップ: 300KB以下が理想的）
- **品質**: 80-85%のJPEG品質で十分です
- **アスペクト比**: 16:9（横長）を推奨します
- **フォーカス**: 画像の中央部分にチームの集合写真が来るように配置してください

## 画像の配置

1. 上記4つの画像ファイルをこのフォルダ（`assets/images/`）に配置してください
2. ファイル名は上記の通りにしてください（大文字小文字は区別されます）
3. 画像を配置した後、ブラウザで確認してください

## イントロアクション用画像

イントロアニメーションで使用する画像ファイルをこのフォルダに配置してください。

以下の5つの画像ファイルを用意してください：

1. **intro-team-1.jpg** - 円陣を組んでいる少年野球チーム（土のグラウンド）
2. **intro-team-2.jpg** - トロフィーと賞状を持った集合写真（屋内）
3. **intro-team-3.jpg** - 一列に並んだ少年たち（屋外、キャッチャー付き）
4. **intro-team-4.jpg** - 後ろ姿の整列（黄色い壁の前、屋内練習場）
5. **intro-team-5.jpg** - 屋外の集合写真（コーチ中心、緑豊かな背景）

### 推奨仕様
- サイズ: 1920x1080px（16:9のアスペクト比）
- 形式: JPEG
- ファイルサイズ: 各画像300KB以下を推奨
- 品質: 80-85%のJPEG品質

## 注意事項

- 画像ファイルが見つからない場合は、フォールバック画像（hero-team.jpg）が使用されます
- すべての画像ファイルが同じアスペクト比であることを推奨します
- 画像の権利を確認し、適切な許可を得てから使用してください
