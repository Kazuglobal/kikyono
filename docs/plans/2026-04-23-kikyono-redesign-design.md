# 桔梗野バイオレッツ リデザイン設計

**日付:** 2026-04-23

## 目的

イントロアニメーションを含め、桔梗野バイオレッツの公式サイトを添付デザインに忠実に合わせて刷新する。
最優先はデスクトップ画面の見た目の一致で、スマホは同じ意匠を保ったレスポンシブ対応とする。

## 承認済みスコープ

- 最初のアニメーションの後に表示されるページ全体を刷新する
- イントロアニメーション自体も本編のブランドトーンに合わせて作り直す
- 再現対象は以下を含む
  - 固定ヘッダー
  - ヒーロー
  - 新入部員募集バナー
  - チーム紹介
  - バリューカード
  - 部員数
  - 大会実績
  - 練習スケジュール
  - 年間スケジュール
  - 保護者の声
  - フッター

## デザイン方針

### 1. 画面構成

- 構成は添付画像の紙面レイアウトを基準にする
- ヘッダーは紫帯ベース、左ロゴ、中央ナビ、右の黄色CTAとする
- ヒーローは集合写真を全面に使い、左側に日本語コピーと2つのCTAを重ねる
- ヒーロー直下に「新入部員大歓迎」の横長バナーを配置する
- 中盤は紹介、価値観、部員数、実績、スケジュールを整理されたカードと表で見せる
- 終盤は保護者の声と情報型フッターで締める

### 2. ビジュアル

- 基調色は濃い紫、白、黄色
- 余白、罫線、カードの角丸、帯見出しを添付画像に寄せる
- 見出しは太く、本文は読みやすい日本語中心の組版にする
- 今の大きな英字ヒーロー見出しは廃止する
- 既存の半透明ダークUIや派手な浮遊表現は削減する

### 3. アニメーション

- イントロは短く上品な導入にする
- 想定演出は紫背景のブランド導入、ロゴと短いコピー、野球ボールまたは光のラインによる1回のアクセント、本編へのフェード接続
- 時間は約2.0秒から2.8秒
- 本編では激しい演出を避け、フェードイン、軽い上昇、CTAホバー程度に抑える

## 実装方針

- 既存の Angular 構成は維持する
- ただし本編のマークアップは `src/app.component.html` を中心に大幅に組み替える
- 既存コンポーネントの役割は以下とする
  - `src/components/intro-animation/*`: イントロ再設計
  - `src/components/header/*`: 添付準拠の固定ヘッダーに刷新
  - `src/components/footer/*`: 情報型フッターに刷新
  - `src/app.component.*`: 本編各セクションを再構築
  - `index.html`: フォント、ベーススタイル、全体トークン、メタ情報の整理

## 制約

- 既存技術スタックは維持する
- Angular 単一ページ構成のまま実装する
- 見た目の忠実再現を最優先し、過剰なコンポーネント分割は行わない
- モバイルでは同一意匠を保ちつつ縦積みと余白の再設計を行う

## 成功条件

- イントロ後の第一印象が添付画像と同系統に見える
- 主要セクションの順序、色、構図、CTA配置が添付に近い
- デスクトップで意図したレイアウトが崩れない
- モバイルでも可読性とCTA到達性を保つ
- `npm run build` が通る
